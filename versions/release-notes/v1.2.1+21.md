# Release Notes

## v1.2.1+21 - 2026-06-12

### 주요 변경
- 홈 위젯의 authoritative refresh, stale food 진행, snapshot 동기화, 스태미나 색상, 캐릭터 표시 크기를 보강했습니다.
- 오프라인 음식 섭취 완료, 부화 시간, 진화 게이지, 질병 확률 등 월드 데이터 진행/밸런스 동작을 정리했습니다.
- 주사기 회복 진동, 임시 상태 아이콘, happy/discover 아이콘, cleanable 점선 테두리 등 인게임 표현을 개선했습니다.
- 스토어 스냅샷 장면 구성과 개발용 메인 씬 시간 배속, Flutter 웹 번들을 갱신했습니다.
