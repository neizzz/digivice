# Release Notes

## v1.2.4+24 - 2026-06-15

### 주요 변경
- 재진입 후 캐릭터 스프라이트시트 cache miss가 발생해도 에셋을 다시 로드해 몬스터 렌더링을 복구하도록 개선했습니다.
- 홈 위젯 background refresh 진단을 보강해 갱신 실패 원인을 더 좁혀 볼 수 있게 했습니다.
- 청소모드와 food eating 중 잘못된 런타임 렌더 참조로 다른 대상이 표시되거나 mask 크기가 어긋나는 문제를 수정했습니다.
- 진화 게이지의 클래스별 목표 시간 편차와 수면 중 진행 페널티를 제거해 진행 정책을 단순화했습니다.
