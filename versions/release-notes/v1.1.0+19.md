# Release Notes

## v1.1.0+19 - 2026-06-08

### 주요 변경
- 홈 위젯 기반을 추가하고 1x1 위젯, 디버그 프리셋, preview/runtime 렌더링, 장기 방치 후 부화 완료 복구, sick/sleep 표시 우선순위를 개선했습니다.
- native lifecycle 기반 world data update와 Flutter bridge 경로를 정리해 앱 재진입과 위젯 갱신의 저장본 동기화 안정성을 높였습니다.
- MonsterInfo와 몬스터북에 gene/진화 정보 표시를 추가하고, 관련 게이지·확률·아이콘 표시를 정리했습니다.
- 알 부화 시간·진화 확률·돌연변이 스택·수면/음식/배변 상태 변화 등 게임 밸런스와 상태 처리 버그를 수정했습니다.
- Play Store/스토어 스냅샷, Flutter web bundle, contributor guide, 내부 debug/asset tooling을 정리했습니다.
