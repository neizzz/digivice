# Release Notes

## v0.5.0+9 - 2026-05-10

### 주요 변경
- 메인씬 UI를 개편해 이름표, 상태 아이콘, 스테미나 및 egg 게이지 배치를 전면 조정했습니다.
- 수면 연출을 텍스트 Z 대신 sleeping 아이콘 기반으로 바꾸고, urgent 상태는 게이지 사이렌 효과로 재구성했습니다.
- 플래피버드 전환 중 back/app hidden 인터럽트 복구와 성능 진단 수집을 강화했습니다.
- 앱이 백그라운드로 가면 플래피버드가 즉시 일시정지되고, 복귀 후에는 설정 메뉴에서 수동으로 재개하도록 바꿨습니다.
- dead 상태 미니게임 진입 시 tomb-in-basket 스프라이트를 사용하고, egg 상태는 dev 모드와 무관하게 미니게임 진입을 막도록 정리했습니다.
- 메인씬 메뉴 광고 기본 쿨다운을 5분에서 2분으로 단축했습니다.

- fix: flappybird 홈 복귀 일시정지 처리
- fix: 수면 Z 이펙트 위치 및 테두리 조정
- refactor: 진입 진단 로그 구조 분리
- fix: flappy debug 로딩 정체 및 로그 수집 개선
- feat: 메인씬 상태 아이콘과 게이지 UI 개편
- feat: 플래피버드 전환 중단과 성능 진단 보강
- chore: 웹 번들과 공개 에셋 산출물 동기화
- feat: 미니게임 진입 상태와 광고 쿨다운 조정
- chore: tomb-in-basket 공개 에셋 동기화
