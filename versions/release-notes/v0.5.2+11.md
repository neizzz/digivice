# Release Notes

## v0.5.2+11 - 2026-05-13

### 주요 변경
- 시간 조작 패널티를 제거하고, 디버그 환경에서 시간 조작 alert를 더 쉽게 확인할 수 있도록 트리거를 추가했습니다.
- dead 상태에서는 미니 게이지가 노출되지 않도록 정리했습니다.
- 청소 모드의 dim overlay와 타겟 렌더링 조건을 보정해 청소 중 표시 안정성을 높였습니다.
- 런타임 시간 기준과 FlappyBird 프레임 처리 흐름을 정리했습니다.
- 캐릭터 이름표 가독성, 화면 이동 경계, 스테미나 색상 구간별 감소율을 조정했습니다.
- virtual bridge web 산출물을 최신 게임 변경 기준으로 다시 동기화하고, 더 이상 쓰지 않는 grass jpg 타일을 제거했습니다.

- fix: 스테미나 색상 구간별 감소율 적용
- fix: 캐릭터 이름표 가독성 조정
- fix: 청소 모드 타겟 렌더링 안정화
- chore: 미사용 grass jpg 타일 제거

- feat: 시간 조작 alert 디버그 트리거 추가
- fix: dead 상태 미니 게이지 숨김
- fix: 시간 조작 패널티 제거
