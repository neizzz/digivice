# Release Notes

## v1.0.4+18 - 2026-05-20

### 주요 변경
- 부트스트랩/설정 흐름에서 뒤로가기 시 setup 화면이 다시 열리는 문제를 막아 초기 진입 안정성을 높였습니다.
- 재진입 시 투척 음식 landed 상태가 어긋나거나 화면/오디오 공존이 불안정하던 문제를 정리해 플레이 연속성을 개선했습니다.
- 몬스터북 진행도 저장, 중복 기록, 중복 진화, reset 이후 유지 동작을 보강해 데이터 일관성을 높였습니다.
- UI 팝 사운드와 효과음 프리로드 경로를 조정하고, 메뉴 광고 기준 및 일부 i18n 크래시를 수정해 전반적인 사용감을 다듬었습니다.
- Android NDK/JVM 타깃 및 Flutter 패키지 버전을 정리해 빌드 기반을 업데이트했습니다.
