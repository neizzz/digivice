# Release Notes

## v1.0.0+14 - 2026-05-18

### 주요 변경
- Setup 이름 입력 안내와 기기 언어 기반 locale 동기화를 정리해 초기 설정 흐름을 안정화했습니다.
- reset popup, native back navigation, 앱 복귀 reentry guard를 보강해 재진입과 뒤로가기 동작을 안정화했습니다.
- 오프라인 광고 대체 화면과 MainScene 광고 pending 상태를 개선했습니다.
- 몬스터북 씬, 진화 카탈로그, 돌연변이 시스템, 희귀도 카탈로그를 추가하고 몬스터북 레이아웃과 현재 몬스터 도달 보정을 조정했습니다.
- 미니게임 음식 상태와 FlappyBird native back 종료 처리를 정리했습니다.
- 모바일 사운드 재생 개선, 게임 효과음/점프 볼륨 조정, 게임오버 버튼 햅틱을 적용했습니다.
- Flutter 웹 번들, i18n 검증 산출물, monster-animations용 슬라임 시트를 갱신했습니다.
