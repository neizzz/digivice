# Release Notes

## v1.2.5+25 - 2026-06-15

### 주요 변경
- 홈 위젯 동기화 시 최신 ECS world data 저장이 완료된 뒤 snapshot을 갱신하도록 순서를 보정해 치료 직후 상태가 되돌아가는 문제를 줄였습니다.
- 홈 위젯이 있는 상태로 앱에 재진입할 때 Flutter lifecycle의 진화 결과, 스태미나, 진화 게이지, 먹이 상태를 반영하도록 수정했습니다.
- 이미 최대 진화 게이지에 도달한 수면 중 저장본도 lifecycle 진행 시 진화되도록 보정했습니다.
- 몬스터 정보 레이어의 진화 게이지 표시를 소수 둘째자리 내림으로 바꿔 100.00%처럼 오해될 수 있는 반올림 표시를 방지했습니다.
- 위젯 재진입 병합 타입 오류를 수정해 릴리즈 빌드 타입체크가 통과하도록 했습니다.
