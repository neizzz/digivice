# Release Notes

## v1.2.0+20 - 2026-06-10

### 주요 변경
- 재진입/백그라운드 복귀 시 수면 상태, 저장 timestamp, 돌연변이 노출 스택이 더 안정적으로 유지되도록 수정했습니다.
- 스토어 스냅샷에 먹는 장면을 추가하고, 관련 캡처/버전 관리 산출물 흐름을 정리했습니다.
- 진화, MonsterBook 저장, 부화 선택 판정을 Flutter lifecycle 권한으로 이전해 앱/위젯 간 상태 판정 기준을 단일화했습니다.
- 위젯 egg 주기 갱신과 foreground hatch update source 타입 오류를 수정했습니다.
- 낮잠 피로도와 게임 밸런스 기준을 정리해 lifecycle 계산 일관성을 높였습니다.
