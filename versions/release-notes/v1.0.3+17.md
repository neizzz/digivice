# Release Notes

## v1.0.3+17 - 2026-05-18

### 주요 변경
- 사운드 체감을 다듬었습니다. 주사기 삽입 효과음의 재생 타이밍과 음원을 손보고, 미니게임 버튼/컨트롤 버튼 효과음 설정과 볼륨 반영을 정리했습니다.
- UI 효과음을 미리 불러오도록 개선해 첫 재생 시 체감 지연을 줄였습니다.
- MainScene에 객체 상한과 청소 경고를 추가해 장시간 플레이 중 상태 관리 흐름을 보강했습니다.
- Android WebView에 hybrid composition을 적용해 표시 안정성을 개선했습니다.

### 포함 커밋
- fix: 주사기 삽입 효과음 타이밍 개선
- feat: UI 효과음 프리로드 추가
- feat: MainScene 객체 상한 및 청소 경고 추가
- fix: 미니게임 버튼 효과음 설정 반영
- fix: 주사기 삽입 효과음 음원 갱신
- fix: Android WebView hybrid composition 적용
- fix: 컨트롤 버튼 효과음 볼륨 감소
