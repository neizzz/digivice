# Release Notes

## v0.6.0+12 - 2026-05-16

### 주요 변경
- FlappyBird 미니게임의 점수 폰트 동기화, game over 점수 표시, 점프 버튼 반응 속도, 오디오 볼륨, 바닥 game over 판정을 개선했습니다.
- MainScene의 감정 표시, 게이지, 상태 아이콘, dead 상태 이름표 배치를 개선했습니다.
- 수면 중 미니게임 진입 시 캐릭터가 깨어나고 피로도/스테미나 패널티가 적용되도록 했습니다.
- 앱 전역 폰트, 설정/팝업/디버그 UI, 다국어 기반과 i18n 스냅샷 캡처 흐름을 정리했습니다.
- Android 디스플레이 모드 제어와 Flutter bridge/광고 SDK 의존성 및 web 빌드 산출물을 갱신했습니다.
- 오프라인 전면광고 대체 화면을 추가했습니다.
- 릴리즈 패키징 자동화 스킬을 추가했습니다.
